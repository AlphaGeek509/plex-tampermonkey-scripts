Lyn-Tron Plex Scripts

TamperMonkey userscripts for Plex QuoteWizard automation.

Prerequisite: Install TamperMonkey in your browser before clicking any Install button.
Quote Tracking

Customer Contact Add
Adds a Hub Bar "New Contact" button on Quote that opens Plex's Contact form in a new tab.
Install

Customer Catalog Get
Watches CustomerNo, fetches Catalog Key/Code, and stores them for use by other scripts.
Install

Part Stock Level Get
Adds a "Get Stock Levels" button on Quote Part Detail and Hub; queries stock and toasts totals.
Install
QT30
Part Catalog Pricing Apply
Applies customer catalog breakpoint pricing, removes zero-qty rows, and sets unit price with rounding.
Install
QT35
Quote Attachments Get
Adds an Attachments badge and button to the Hub Bar; counts and displays document attachments per quote.
Install
QT50
Quote Validation
Runs rule-based checks on quote lines for lead time, unit price limits, and part number management. Adds a "Validate Lines" button with a details modal and CSV export.
Install
Scripts update automatically via TamperMonkey (checks every 6 hours).
